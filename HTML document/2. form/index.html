Kurs tworzenia stron internetowych

Aby rozpocząć, nie potrzebujesz żadnej wiedzy o programowaniu czy działaniu internetu — wszystkiego Cię nauczymy! Przeprowadzimy Cię przez wszystkie niezbędne rzeczy, których potrzeba do stworzenia Twojej własnej strony internetowej i pokażemy Ci jak umieścić ją w internecie.

Po ukończeniu kursu zdobędziesz takie umiejętności jak:

  • rozumienie składni i struktury HTML
  • łączenie HTML z CSS i JavaScript
  • Responsive Web Design, czyli tworzenie stron, które automatycznie dostosowują się do rozdzielczości ekranu użytkownika
  • przygotowanie grafik na potrzeby Twojej własnej strony
  • wdrażanie opcji dostępności dla osób z niepełnosprawnościami

Każda osoba, która ukończy kurs i wykona projekt końcowy otrzyma certyfikat potwierdzający nabyte umiejętności.
